About

On this app, every day people can rent out their backyards to others as intimate, inexpensive wedding venues. Similar to how one would rent out their living space on AirBnB.

Frontend

Built with Angular. Deployed on AWS S3.

Backend

Powered by Java with Maven - Spring Boot. Deployed on AWS Elastic Beanstalk.

Database

MySQL database hosted on AWS RDS. Includes Customer, Partner, Backyard and Event tables, where a Customer and Backyard can have many events respectively, and a Partner can have many Backyards.
